# 急速発酵乾燥による有機廃棄物処理・有機肥料化 — プレゼン資料＆動画

パンパンガ州 廃棄物管理会合（2026年 第2四半期）での技術紹介（「固形廃棄物管理技術の
ご紹介」枠、約30分）を想定した、スライド資料・動画・台本一式です。
**日本語 / English / Tagalog（Filipino）** の3言語で用意しています。

## 成果物（`output/<言語>/`）

| ファイル | 内容 |
|---|---|
| `deck_<lang>.pdf` | スライド資料（16:9・全25ページ）|
| `video1_technology_<lang>.mp4` | 動画①「急速発酵乾燥技術のしくみ」（約1分27秒）|
| `video1_technology_<lang>.srt` | 動画①の字幕（読み上げ原稿）|
| `video2_circular_model_<lang>.mp4` | 動画②「循環型GX農業モデル」（約1分23秒）|
| `video2_circular_model_<lang>.srt` | 動画②の字幕（読み上げ原稿）|
| `script_<lang>.md` | 司会・登壇者用の台本（30分の進行）|

`<lang>` = `ja`（日本語）/ `en`（英語）/ `tl`（タガログ語）

## 30分の構成（スライドと動画を交える設計）

1. 導入（約1分）— スライド
2. 会社紹介（約4分）— スライド
3. 背景と課題（約4分）— スライド
4. **▶ 動画①** 急速発酵乾燥技術のしくみ（約1.5分）
5. 技術の詳細と処理プロセス（約6分）— スライド
6. GHG削減効果の検証 / アンヘレス市（約4分）— スライド
7. **▶ 動画②** 循環型GX農業モデル（約1.5分）
8. FS事業の計画とロードマップ（約4分）— スライド
9. まとめ（約1分）— スライド

スライド資料の中に「▶ 動画①」「▶ 動画②」の差し込みページがあり、その位置で
該当の動画を再生する流れです。動画の読み上げ原稿は `.srt` 字幕および各動画の進行に
対応しています（必要に応じて、お手元で吹替録音に利用できます）。

## 動画の音声について

本制作環境では外部の音声合成サービス（Google / Microsoft / HuggingFace）への接続が
ネットワークポリシーで遮断されているため、自動ナレーション音声は付与していません。
動画は画面内の解説テキストで内容を説明する構成とし、各言語の `.srt` 字幕と
読み上げ原稿（`script_<lang>.md` および各シーンの字幕）を併せて納品しています。

## 内容の方針（依頼に基づく制約）

- 出力物の用途は **有機肥料に限定**。プラスチック等の不適物は工程内で **自動的に分別**
  される旨にとどめています（燃料化等の用途には言及していません）。
- GHG削減量は **埋立メタンの回避** による効果（年間 約2,016〜2,352 t-CO₂e）を提示。
- 装置メーカー名やその他の関係団体名は記載していません。
- 参加者の名指しや協力のお願いは含めず、技術紹介に徹した中立的な内容です。

## 再生成（任意）

```bash
cd presentation
pip install Pillow imageio-ffmpeg
python3 build.py        # output/ 以下に全成果物を再生成
```

- `engine.py` … 描画エンジン（PIL）。スライドと動画フレームの共通レンダラ。
- `content_ja.py` / `content_en.py` / `content_tl.py` … 言語別の本文データ。
- `build.py` … PDF・MP4・SRT・台本を一括生成。
